## 📚 Стек

- **Основы Python**: синтаксис, переменные, типы данных, условные операторы, циклы, функции.
- **Работа с файлами и модулями**: чтение/запись файлов, создание и использование модулей.
- **ООП (Объектно-Ориентированное Программирование)**: классы, наследование, инкапсуляция, полиморфизм.
- **Работа с библиотеками**: `json`, `requests`, `datetime`, `logging`, `regex`, `collections`, `random`, `pandas`, `csv`.
- **Виртуальные окружения и управление зависимостями**: `venv`, `poetry`.
- **Контроль версий**: Git, GitHub.
- **Тестирование**: pytest, фикстуры, параметризация, unit‑тесты, логирование.
- **Работа с базами данных**: основы SQL, использование SQLite и PostgreSQL.
- **Разработка на Django**: основы Django, шаблонизация, ORM, FBV & CBV, аутентификация и права доступа, формы, админка, веб-приложение).
- **Кэширование**: Redis.
- **Фронтенд‑интеграция**: HTML/CSS/JS.
- **Безопасность**: CSRF.
- **Контейниризация приложений**: Docker, Docker Compose
- **Развертывание и непрерывная доставка на сервер**: Nginx, Gunicorn, CI/CD
- **Асинхронная обработка задач**: Celery, Celery-beat
- **Автоматизированное тестирование интерфейса браузера**: Selenium

## 💼 Проекты

- **Анализ банковских транзакций**: приложение для анализа и визуализации данных из Excel-файлов.
- **Поиск вакансий**: приложение для поиска и выборки вакансий с использованием API hh.ru и библиотеки requests.
- **Поиск вакансий с подключением БД**: приложение для поиска данных о вакансиях с сайта hh.ru с подключением БД PostgreSQL.
- **Сервис управления рассылками**: приложения для создания автоматической рассылки со сбором статистики.
- **Образовательная платформа**: приложение для размещения и оплаты образовательных материалов с интеграцией оплаты Stripe.
- **Трекер привычек**: приложение для отслеживания полезных привычек с интеграцией Telegram-бота.
- **Парсер средней стоимости товаров**: приложение для расчета средней стоимости выбранных строительных товаров с полным циклом развертывания и автоматической доставки на сервер.
